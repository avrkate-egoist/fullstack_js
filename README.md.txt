# Портфоліо Катерини

Вітаю! Це моє персональне портфоліо, яке демонструє мої навички у дизайні та веброзробці.

## Опис проекту

Проект представляє мої роботи у трьох основних напрямах:

1. **2D-графіка** – створення макетів для лазерного гравіювання, вишивки, друку та дизайнів для патчів і шевронів.  
2. **3D-дизайн** – моделювання пропсів для ігор, концептів та візуалізацій з використанням Blender, ZBrush, Substance Painter та Marmoset Toolbag.  
3. **Веброзробка** – розробка адаптивних сторінок і інтерфейсів за допомогою HTML, CSS та JavaScript.

## Функціональні можливості

- Галерея робіт з прикладами 2D-макетів, 3D-пропсів та вебсторінок.
- Сторінка сертифікатів із підтвердженням проходження курсів.
- Контактна форма для зворотного зв’язку.
- Навігаційні посилання (якорі) для швидкого переходу між секціями.

## Освіта та професійний розвиток

- Навчання у **Київському національному університеті технологій та дизайну (КНУТД)**.
- Курси з 3D-моделювання (Autodesk Maya, Blender, Substance Painter, ZBrush, Marmoset Toolbag, Unreal Engine).  
- Full Stack JavaScript: фронтенд і бекенд, створення вебзастосунків.

## Досвід роботи

- **2D-графіка** – макети для патчів, шевронів, лазерного гравіювання та друку.  
- **3D-дизайн** – моделювання об’єктів для ігор та візуалізацій.  
- **Веброзробка** – створення адаптивних вебсторінок та інтерфейсів.

Працювала з невеликими студіями та на фрілансі, брала участь у командних проектах у різних сферах.

## Підхід до роботи

Поєдную аналітичне мислення з художнім баченням. Створюю дизайн, який одночасно естетичний, функціональний та емоційно зрозумілий користувачеві.
